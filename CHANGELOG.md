# Changelog

All notable changes to **Reverse Software Search (RSS)** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-23

### Added
- **Multi-Agent Engineering Council Engine**: 4 AI persona reviewers (*Architect*, *FinOps*, *Security*, *UX*).
- **Live WebContainer Code Sandbox**: In-browser Node.js/Python virtual process runner and code explorer.
- **Preset Creative Synthesizer Templates**: Pre-loaded prompts for Datadog alternative, API bridge generator, traffic simulator, and repo scanner.
- **GitHub Actions CI & GitHub Pages Deployment**: Automated static export and dual-mode publishing to GitHub Pages.
- **Security Headers & Telemetry API**: OWASP security headers middleware and `/api/v1/health` endpoint.
- **Pytest Suite**: Backend unit tests covering sub-engines and models.
